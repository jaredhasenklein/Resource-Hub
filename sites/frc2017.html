Web Based Calculator

This version of the calculator receives instantaneous updates and can be used right on this page in your browser. Some school firewalls and outdated browsers/plugins  may block it.

Google Sheets Calculator

This version of the calculator will stop receiving updates once you make a copy of the sheet. We recommend you check the changelog on the right and make a new copy after any changes. Anyone with a Google Account can use this version. Button not working? Click here!

Offline Download Calculator

The offline download is an Excel (.xlsx) file. This version is best for those without regular internet access. but will stop receiving updates once you download. We recommend you check the changelog on the right and download a new copy after any changes. This has been tested with the latest several versions on Excel on Windows and Mac and has not been tested with Numbers or Open Office. Button not working? Click here!
