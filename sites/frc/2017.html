Scoring Calculator

Use the calculator to calculate the values of the missions.
  • Google Sheets (recommended): The calculator is designed for Google Sheets and best used on that platform. this requires a Google Account (free) and you'll need to make a fresh copy of the sheet if we post any updates. Make a copy of Google Sheets
  • Excel download (best for offline): You can download the calculator and use it in Excel. This requires Microsoft Office (costs money if you don't already have the suite with Word, PowerPoint, etc.) and you'll need to download the file again if there are updates. Download Excel
  • Use on this site (beta): You can also use the calculator straight on this website. You don't need any accounts or software and won't need to update it. However, this is not perfected software and might break at times. Use with caution. Load calculator below
