Slack Rules Bot

A bot for looking up FRC rules via a slash command in Slack

Home > Resource Hub Documents

Early development - limitations apply (PLEASE READ THIS BOX!)

At the moment, this bot is a proof of concept. There are several limitations to note:

  • Formatting issues will apply (such as missing spaces or boxes in place of special characters)
  • Only rules starting with R (e.g. R30) will be avalible for now. Fouls, other rules, and Q&A are coming soon
  • Certain rules may be missing. We are working to fix this but it may take some time.
  • This only works in channels, not DMs.
  • Invalid inquiries will not get a response - yet. This will be fixed in a future update.
Give feedback

Installation Click the button to Add to Slack. Authorize the app (you may need to be an admin depending on workspace settings. Then, send /rules r# (e.g. /rules r30) in any channel (not in a DM). Please read the limitations above before using. This is a proof of concept only and will have significant limitations.
