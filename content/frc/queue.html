Lead Queuing Tool

Home > Resource Hub Documents
A spreadsheet-based tool to help Lead Queuers manage the robot and match flow at events. Make a copy

Features

  • Keep track of the whereabouts of each team
  • Visually identify the pit locations of teams needed for upcoming matches
  • Automatically generate playoff match information, before the FMS
  • Optionally share data with queuing staff

Requirements

  • One (1) handheld device with internet access (mobile data, if possible since event WiFi is unpredictable)
  • A Google account for the Lead Queuer (free)
  • If using an Apple or Android tablet or phone, install the "Google Sheets" app
  • Access to the pit map prior to the event preferred

Setup and usage

Prior to the event
  1. Make a copy of the Google Sheet using the link above.
  2. Configure the pit map on the "Map" tab according to the instructions once it is available
    • Tip: If the pit layout isn't ready, ask event organizers if they at least have a blank version. This will let you create the structure which takes the most time. You can always input the team list later.
  3. Once your path is established for teams, go to the "Configure" tab to create options for where teams can be. There's a good list of common choices already there. You can hide the sheet when you're done.
  4. Add the match schedule to the ScheduleSetup tab using the defined format. Entering your event code in the designated box will automatically provide you with a link to the match schedule to copy and paste.
    • Note: This schedule usually isn't ready until late the night before or early the first morning of qualification matches. Just make sure you can access a device at some point after the match schedule is released before the first matches. A computer is recommended for this step.

During the event
  1. During qualification matches, the Lead Queuer will enter the status of teams onto the "MASTER QUEUING STATUS" tab.
    • Optionally, share the document with your queuers (only give them "View Only" permissions) so they can check the status without asking you.
  2. During alliance selection, enter the data as it becomes ready on the "Alliances tab"
  3. During playoff matches, switch to the "Playoffs" tab. Do the same thing as qualification matches, but be sure to select the winner of each match to generate the schedule for the upcoming matches. Entering everything in real time will ensure you have the schedule as fast as possible.
    • Since the schedule always follows the same format with a bracket, you can know which alliances will play which other alliances and what color each alliance will be before the schedule is even posted. This is very helpful in the fast-paced elimination rounds.
